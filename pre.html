what is it

the de-link pre is prototype 0 of the de-link platform. not a product. not the goal. it proved that an open, hand-solderable, 3D-printable e-paper reader could actually work. now development moves toward v1.0: pip.

easy to build, easier to repair

0805 and SOT-23 components throughout. hand-solderable with basic equipment. the enclosure is fully 3D-printable. when something breaks, you replace the part. want a different screen? get a new panel and print a new case.

software freedom

flash crosspoint reader over USB-C and you have a working e-reader in minutes. no locked bootloader. no bricks. write your own firmware if you want.

Hardware Features
processor
dual-core ESP32-S3 at 240 MHz. headroom to run display refresh and background tasks simultaneously.
storage
4-bit SDMMC for fast microSD reads. epub page streaming is noticeably quicker than with single-bit SPI.
frontlight
hardware series LED driver on-board. software-controlled cool/warm temperature. no external accessory needed.
input
two independent 4-switch resistor ladders plus a multi-function GPIO and reset.
USB-C + OTG
USB OTG supports file transfer over USB-C without removing the SD card. works as a USB device, not just a charger port.
battery
optional onboard overcharge/discharge protection. accepts almost any single-cell 3.7V LiPo with a JST 2.0 connector.
screen
any 24-pin GoodDisplay SPI e-paper panel. 3.97", 4.26", 7.5", and more. swap the screen without touching the PCB.
What the pre gets wrong

this is a first prototype. it does everything it is supposed to do and i use mine every day. but the shape is a compromise, the enclosure is bulkier than it needs to be, and the per-unit build cost is higher than i want it to be long-term.

the PCB layout has room to shrink. the case has room to get thinner. the BOM has room to get cheaper. the pre proves the idea works. pip is where it becomes something worth recommending to someone else.

what comes next

pip is v1.0. PCB design is nearly final. build guides, sourcing tutorials, and community software are coming. learn more about where the platform is headed.
